## Quick Start

1. **Select Target Chip or Development Board**
   ```bash
   cd apps/games/[game_name]
   tos.py config choice

   ```

2. Configure Game-Specific Parameters (Optional)
   ```bash
   tos.py config menu  
   ```

3. **Build and Flash**
   ```bash
   tos.py build
   tos.py flash
   ```
