## 快速开始

1. **选择目标芯片或开发板**
   ```bash
   cd apps/games/[game_name]
   tos.py config choice

   ```

2. 配置游戏特定参数（可选）
   ```bash
   tos.py config menu  
   ```

3. **编译和烧录**
   ```bash
   tos.py build
   tos.py flash
   ```
